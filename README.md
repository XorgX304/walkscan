<i class="icon-puzzle"></i> **Usage**
-------------
Optional Arguments:

	$ pyscan.py [--h HELP] [-h HOST] [-w WRITE] 
	
	
	-h,HELP, --help 	Show this help message and exit
	-h HOST, --host   	Enter host for port scan
	-w WRITE, --write 	Save this result on file
